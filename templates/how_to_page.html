Before You Start

  • Confirm your email and telephone number are up to date.
  • Install an authenticator app if possible.
  • Prepare a secure location for backup codes.
Emergency Checklist

Step-by-Step Guide

1

Open Security Settings

Go to account settings, then select the security or sign-in section.

2

Select Two-Factor Authentication

Choose authentication app when available. SMS can be fallback only.

3

Verify and Save Recovery Codes

Complete verification, then store one-time backup codes in your vault notes.

4

Test a New Login

Log out and sign back in once to confirm 2FA works before closing settings.

Authentication Methods

Method Convenience Security Recommended Use
Authenticator App Medium High Default choice for most users
SMS Code High Medium Fallback if app setup is not possible
Hardware Key Medium Very High High-risk and admin accounts
